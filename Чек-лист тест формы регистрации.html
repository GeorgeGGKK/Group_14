Чек-лист формы регистрации.

Вопросы для уточнения:

  1. Для чего эта форма?
  2. Какие поля являются обязательными?
  3. Какой язык ввода?
  4. Какое кол-во символов в каждом поле?
  5. Можно ли в поля вводить спец. символы и цифры?
  6. Есть ли требование что пароль будет закрашиваться точками?
  7. Где поле подтверждение пароля?
  8. Какая реакция валидации на пустые поля или некорректные символы?
  9. Присутствуют ошибки в словах.
  10. В зеленой кнопке в конце надпись “Sudmit” не по центру.

Данные для формы:

  1. Язык ввода - латиница.
  2. Кол-во символов для 1 и 2 поля - от 10 до 128
    4 поле - от 8 до 24
  3. 1 и 2 поле текстовые.
  4. Обязательные поля почта и пароль.
  5. В 4 поле нельзя вводить спец. символы, только буквы Верхнего, нижнего регистра и цифры.
  6. Почта - проверка маски( слово-@-домен-.-доменная зона)

Тестирование обязательных полей

  1. Все поля оставляем пустыми
  2. Вводим валидный email
  3. Вводим email без @
  4. Вводим email с двумя @@
  5. Вводим email с некорректным доменом
  6. Email с некорректной доменной зоной
  7. Email со спец символами в слове
  8. В поле password вводим валидные данные
  9. password с буквами верхнего регистра
  10. password с буквами нижнего регистра
  11. password с буквами верхнего и нижнего регистра
  12. password с буквами верхнего регистра и цифры
  13. password с буквами нижнего регистра и цифры
  14. password с буквами верхнего, нижнего регистра и цифры
  15. password с буквами верхнего, нижнего регистра, цифры и спец. символы
  16. password с буквами верхнего регистра, цифры и спец. символы
  17. password с буквами нижнего регистра, цифры и спец. символы
  18. password с буквами верхнего, нижнего регистра и спец. символы
  19. password с буквами верхнего регистра и спец. символы
  20. password с буквами нижнего регистра и спец. символы
  21. password ввести 7 букв
  22. password ввести 9 букв
  23. password ввести 23 буквы
  24. password ввести 25 букв
  25. Зарегистрировать пользователя который ранее был зарегистрирован

Тестирование необязательных полей (1 и 2 поле)

  1. Name ввести кириллицей
  2. Name ввести кириллицей и латиницей
  3. Name ввести 9 букв
  4. Name ввести 11 букв
  5. Name ввести 127 букв
  6. Name ввести 129 букв
  7. Name ввести цифры
  8. Name ввести буквы с цифрами
  9. Name ввести буквы,цифры,буквы(dawadwa123555sadwqe)
  10. Name ввести буквы,цифры,спец. символы
  11. Name ввести буквы и спец. символы
  12. Name ввести цифры и спец. символы
  13. Name ввести спец. символы
  14. Name ввести смайлики
